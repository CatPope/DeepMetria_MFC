## 1. 프로젝트 개요 ⏱ 1:15

> 🎯 "내 얘기네"라는 공감을 먼저 만든다

**1-1. 문제**
- 마케터·기획자·경영진은 매일 데이터를 보지만, 직접 분석은 막힌다
- 엑셀 함수·피벗을 모르면 멈추고, 분석은 늘 개발자에게 부탁 → 며칠 대기
- 🎤 "데이터는 누구나 갖고 있는데, 분석할 수 있는 사람은 따로 있습니다."

**1-2. 해결책 한 문장**
- **"파일을 올리고, 말로 질문하면, AI가 차트로 답한다."**
- 📁 불러오기 → 🤖 자동 요약 → 💬 자연어 질문 → 📊 차트
- 코드·전문지식 없이, 내 PC에서 쓰는 데스크톱 앱

🎨 막힌 사용자 일러스트 + 4단계 가로 플로우

## 2. 프로젝트 수행 — 무엇을 만들었나 ⏱ 1:30

> 🎯 기술 설명이 아니라 "어떤 점이 좋은가"를 비유로

**2-1. 강점 ① 배울 필요가 없다**
- 평소 동료에게 묻듯 질문하면 끝
- 예: "지역별 매출 1위는?", "지난달 대비 얼마나 늘었어?", "이상한 값 찾아줘"

**2-2. 강점 ② AI가 숫자를 지어내지 않는다 (가장 중요)**
- 🎤 "ChatGPT에 숫자를 물으면 가끔 틀린 답을 그럴듯하게 지어냅니다. 그게 가장 불안한 점이죠."
- DeepMetria는 AI가 **직접 계산하지 않습니다.** AI는 '어떻게 분석할지'만 정하고, 실제 계산은 **미리 검증해 둔 계산기(분석 도구)** 가 합니다
- 비유: AI는 똑똑한 **비서**, 계산은 정확한 **계산기** → 그래서 숫자를 믿을 수 있다

**2-3. 강점 ③ 내 데이터는 안전하다**
- 데이터는 내 PC에 저장되고, 통째로 외부에 올리지 않는다
- 접속 키는 암호화되어 보관된다
- 🎤 "회사 데이터가 새는 거 아니냐는 걱정, 그 부분을 신경 썼습니다."

🎨 "비서(AI) ↔ 계산기(검증된 도구)" 그림 + 자물쇠 아이콘

## 3. 실행 결과 — 직접 보는 데모 ⏱ 2:15 (중심)

> 🎯 백문이 불여일견. 발표의 하이라이트.

**3-1. 데모 (4단계)**
1. 매출 데이터 파일(엑셀)을 연다
2. AI가 자동으로 요약을 만든다 (어떤 데이터인지, 핵심 숫자)
3. "월별 매출 추이 보여줘"라고 입력한다
4. 막대/선 차트가 자동으로 만들어진다
- 🎤 "함수도, 차트 설정도 없습니다. 말만 하면 됩니다."

**3-2. 무엇이 되나 (한눈에)**
- ✅ 엑셀/CSV 파일 불러와 자동 요약
- ✅ 말로 질문해서 차트 만들기
- ✅ 차트 크기·색 편집하고 이미지로 저장
- ✅ 여러 AI(Claude·GPT·Gemini) 중 선택 사용
