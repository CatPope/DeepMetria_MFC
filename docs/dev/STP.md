## 1. 시장 세분화 (Segmentation)

### 세분화 변수

DeepMetria의 핵심 가치는 "코드 없이 데이터 분석"이다. 세분화는 **역할·기술 수준·Windows 환경 의존도**를 기준으로 한다.

### 세그먼트 맵

```
               데이터 활용 빈도 높음
                      │
       기업 내부팀  ───┼─── 컨설팅/에이전시
      (마케팅/운영)    │       팀
            │          │         │
낮은  ──────┼──────────┼─────────┼────── 높은
기술역량    │          │         │       기술역량
            │          │         │
       중소기업  ─────┼─── 연구자/
       대표/임원        │   개인 분석가
                      │
               데이터 활용 빈도 낮음
```

### 세그먼트 프로필

#### 세그먼트 A: 중소기업 마케팅·운영팀 (핵심)

| 항목 | 내용 |
|------|------|
| 기업 규모 | 10~200명 |
| 역할 | 마케팅 담당자, 운영 매니저, 기획자 |
| 기술 수준 | 엑셀 수준, SQL 모름 |
| 예산 | 라이선스 1회 10~50만 원 또는 연간 유지보수 계약 |
| 데이터 | 매출, 광고 성과, 고객 데이터 (로컬 파일 기반) |
| 주요 니즈 | 빠른 인사이트 도출, 보고서 자동화, 오프라인 사용 |
| 구매 방식 | 팀장/대표 결재, 견적서 기반 라이선스 구매 |

**해결할 Jobs to Be Done:**
- 사내 보안 정책으로 클라우드 업로드 없이 데이터를 분석하고 싶다
- 데이터팀 없이도 근거 있는 의사결정을 하고 싶다
- 임원 보고용 시각화 자료를 빠르게 만들고 싶다

**Pain Points:**
- 클라우드 SaaS에 민감 데이터 업로드 불가 (보안 정책)
- 엑셀 피벗이 한계에 달했을 때 대안 없음
- BI 툴(Tableau Desktop, Power BI)은 너무 복잡하고 라이선스 비쌈

---

#### 세그먼트 B: 컨설팅·에이전시 팀 (성장)

| 항목 | 내용 |
|------|------|
| 기업 유형 | 경영 컨설팅, 마케팅 에이전시, 리서치 기관 |
| 기업 규모 | 5~100명 |
| 역할 | 컨설턴트, 애널리스트, 어카운트 매니저 |
| 기술 수준 | 중간 (엑셀 고급, 일부 SQL) |
| 예산 | 좌석당 라이선스 또는 팀 라이선스 20~100만 원 |
| 주요 니즈 | 클라이언트별 분석 속도, 발표 자료 품질, 오프라인 납품 |

**Jobs to Be Done:**
- 클라이언트 NDA 데이터를 외부 서버 없이 안전하게 분석하고 싶다
- PPT 발표용 차트를 자동 생성하고 싶다
- 여러 클라이언트를 동시에 효율적으로 관리하고 싶다

**Pain Points:**
- 클라이언트 데이터를 클라우드에 올릴 수 없는 계약 제약
- 마감 압박 속에서 분석 품질 유지 어려움
- 인력 대비 프로젝트 수 증가로 생산성 한계

---

#### 세그먼트 C: 연구자·개인 분석가 (볼륨)

| 항목 | 내용 |
|------|------|
| 유형 | 대학원생, 연구원, 프리랜서 데이터 분석가, 재무 담당자 |
| 규모 | 1~3인 |
| 기술 수준 | 중간 (엑셀 고급, 통계 기초) |
| 예산 | 개인 라이선스 5~20만 원 (일회성) |
| 주요 니즈 | 셀프서비스, 즉각적 결과, 로컬 실행 |

**Jobs to Be Done:**
- 인터넷 없이도 현장에서 데이터를 분석하고 싶다
- 복잡한 툴 설정 없이 바로 쓸 수 있는 분석 도구가 필요하다

**Pain Points:**
- 클라우드 구독 비용 부담 (월정액보다 일회성 구매 선호)
- 보안 네트워크 환경(군, 병원, 공공기관)에서 인터넷 접근 제한
- 분석 결과를 어떻게 해석해야 할지 모름

---

#### 세그먼트 D: 공공기관·금융·의료 규제 산업 (미래)

| 항목 | 내용 |
|------|------|
| 기관 유형 | 공공기관, 은행, 보험, 병원 |
| 규모 | 100명 이상 |
| 역할 | 데이터 담당자, 기획부서, 내부 감사 |
| 기술 수준 | 다양 (비개발자 중심) |
| 예산 | 기관 단위 라이선스 200만 원 이상 |
| 주요 니즈 | 완전한 온프레미스 실행, 감사 로그, 데이터 외부 유출 제로 |

**Pain Points:**
- 규제상 클라우드 전송 불가 (개인정보보호법, 금융보안 규정)
- 기존 데스크톱 툴(Excel 애드인)의 분석 한계
- 도입 시 IT 부서 승인 필요 → 설치형 소프트웨어 선호

| 세그먼트 | 강점 | 약점 |
|----------|------|------|
| A 중소기업팀 | 명확한 Pain(보안), 높은 WTP, 팀 단위 라이선스 | 초기 영업 사이클 존재 |
| B 에이전시 | 멀티시트 확장, 납품 레퍼런스 효과 | 프로젝트 종료 시 갱신 동기 약화 |

| D 규제 산업 | 높은 단가, 장기 계약, 교체 장벽 높음 | 긴 도입 승인 기간, 인증 요구 |

### 타겟 전략: 집중화 → 차별화 순차 확장

**MVP 1차 타겟 (집중화): 세그먼트 A — 중소기업 마케팅·운영팀**
- DeepMetria 핵심 가치(로컬 AI 분석, 보안 데이터 처리)와 가장 높은 적합성
- 설치 후 즉시 사용 가능, 빠른 구매 결정
- Standard/Pro 라이선스 업그레이드 경로가 명확

**2차 타겟 (확장): 세그먼트 B — 컨설팅·에이전시 팀**
- 팀 라이선스로 ARPU 극대화
- 고객사 납품 레퍼런스를 통한 신규 고객 유입 기대

**서비스하되 집중하지 않음: 세그먼트 C**
- Trial 라이선스로 유입, 정식 구매 전환 유도
- 리소스 집중 대상 아님

**보류: 세그먼트 D**
- MVP 이후 Enterprise 라이선스 안정화 후 진입
- 공공조달 등록, 보안 인증(CC, ISMS) 획득 후 공략

## 3. 포지셔닝 (Positioning)

### 경쟁 포지셔닝 맵

```
                  사용 편의성 (높음)
                        │
           DeepMetria   │
           (목표)       │   Excel 애드인
                        │   (범용형)
                        │
비전문가 ───────────────┼─────────────── 전문가
친화적                  │               친화적
                        │
           SPSS/SAS     │   Tableau Desktop/
           (구형 레거시) │   Power BI Desktop
                        │
                  사용 편의성 (낮음)
              (분석 전문성 요구)
```

### 포지셔닝 옵션

**옵션 1: 보안을 지키는 AI 데이터 분석가**
```
데이터를 분석하고 싶지만 클라우드 업로드가 불가능한 팀을 위해,
DeepMetria는 완전 로컬 실행 LLM 기반 데이터 분석 데스크톱 앱으로,
인터넷 없이도 전문가 수준의 분석과 시각화를 제공합니다.
Tableau Desktop이나 Power BI와 달리,
저희는 코드나 SQL 없이 대화하듯 데이터를 분석하게 합니다.
```

**옵션 2: 오프라인 AI 데이터 팀장**
```
데이터 분석가를 고용할 여력이 없는 Windows 기반 팀을 위해,
DeepMetria는 설치형 AI 분석 소프트웨어로,
파일 열기부터 대시보드, 발표 자료까지 오프라인 원스톱으로 처리합니다.
엑셀 매크로나 SPSS와 달리,
저희는 분석 도구를 자동 선택하고 맥락 있는 인사이트를 제공합니다.
```

**옵션 3: 로컬 우선 분석의 민주화**
```
데이터가 외부로 나가는 것이 걱정되는 팀을 위해,
DeepMetria는 PC에서만 실행되는 AI 분석 소프트웨어로,
"왜 매출이 떨어졌나요?"처럼 자연어로 물으면 답을 드립니다.
복잡한 BI 툴과 달리,
저희는 설치 후 5분 안에 첫 인사이트를 경험할 수 있습니다.
```

### 최종 포지셔닝 권고

**1차 타겟(중소기업팀) 기준 권고 포지셔닝:**

> **DeepMetria는 보안이 중요한 팀을 위한 로컬 AI 데이터 분석 소프트웨어입니다.**
>
> 데이터를 열고 궁금한 것을 물어보면, AI가 알아서 분석하고 시각화합니다.
> SQL도, 코딩도, 인터넷도 필요 없습니다.
>
> - 파일 열기 → 자동 데이터 요약
> - 자연어 질문 → 적합한 분석 + 차트 자동 선택
> - PPT처럼 편집 가능한 대시보드 → 바로 내보내기
>
> Tableau Desktop은 너무 어렵고, Excel은 분석 깊이가 부족합니다.
> DeepMetria는 그 사이를 채웁니다. 데이터는 당신의 PC를 떠나지 않습니다.

### 포지셔닝 증거 포인트

| 주장 | 증거/지표 |
|------|----------|
| 비개발자도 즉시 사용 가능 | 설치 후 5분 내 첫 인사이트 달성 |
| 완전 로컬 실행 | 분석 데이터 외부 전송 없음, 오프라인 동작 |
| 자동 분석 도구 선택 | 사용자가 차트 유형을 지정할 필요 없음 |
| 발표 자료 즉시 활용 | PPT/PDF 내보내기, 서식 편집 지원 |
| 깊이 있는 분석 | Chain of Thinking 기반 다단계 추론 |
| 멀티 LLM 지원 | 로컬 모델(Ollama) 및 선택적 외부 API 지원 |

### 라이선스 티어별 포지셔닝

| 티어 | 타겟 | 핵심 메시지 |
|------|------|------------|
| Trial | 연구자, 개인 | "일단 무료로 설치해서 써보세요" |
| Standard | 중소기업 담당자 | "팀의 데이터 분석을 자동화하세요" |
| Pro | 에이전시, 다수 프로젝트 | "클라이언트마다 전문 분석 보고서를" |
| Enterprise | 공공기관, 규제 산업 | "보안 검증된 온프레미스 AI 분석" |

## 4. 실행 요약

### STP 결정 요약

| 항목 | 결정 |
|------|------|
| **1차 타겟 세그먼트** | 중소기업 마케팅·운영팀 (비개발자, 보안 민감) |
| **2차 타겟 세그먼트** | 컨설팅·에이전시 팀 |
| **타겟 전략** | 집중화 → 차별화 순차 확장 |
| **핵심 포지셔닝** | "보안이 중요한 팀을 위한 로컬 AI 데이터 분석 소프트웨어" |
| **핵심 차별점** | 완전 로컬 실행 + 자연어 질문 → 자동 분석 + 시각화 원스톱 |
| **경쟁 우위** | BI 툴보다 쉽고, 클라우드 SaaS보다 안전하며, Excel보다 강력 |
| **MVP 진입점** | Trial 라이선스 무료 배포 → Standard 전환 |

### 다음 단계

1. **ICP(Ideal Customer Profile) 구체화** — 보안 민감 중소기업 마케팅팀 3~5개사 인터뷰
2. **포지셔닝 검증** — 랜딩 페이지 A/B 테스트 (옵션 1 vs 옵션 3)
