## 1. 시장 세분화 (Segmentation)

### 세분화 변수

DeepMetria의 핵심 가치는 "코드 없이 데이터 분석"이다. 세분화는 **역할·기술 수준·데이터 활용 목적**을 기준으로 한다.

### 세그먼트 맵

```
               데이터 활용 빈도 높음
                      │
       기업 내부팀  ───┼─── 컨설팅/에이전시
      (마케팅/운영)    │       팀
            │          │         │
낮은  ──────┼──────────┼─────────┼────── 높은
기술역량    │          │         │       기술역량
            │          │         │
       중소기업  ─────┼─── 1인 사업자/
       대표/임원        │   프리랜서
                      │
               데이터 활용 빈도 낮음
```

### 세그먼트 프로필

#### 세그먼트 A: 중소기업 마케팅·운영팀 (핵심)

| 항목 | 내용 |
|------|------|
| 기업 규모 | 10~200명 |
| 역할 | 마케팅 담당자, 운영 매니저, 기획자 |
| 기술 수준 | 엑셀/구글 시트 수준, SQL 모름 |
| 예산 | 월 5~50만 원 수준 SaaS 구독 |
| 데이터 | 매출, 광고 성과, 고객 데이터 |
| 주요 니즈 | 빠른 인사이트 도출, 보고서 자동화 |
| 구매 방식 | 팀장/대표 결재, 빠른 의사결정 |

**해결할 Jobs to Be Done:**
- 데이터를 보고서로 만드는 시간을 줄이고 싶다
- 데이터팀 없이도 근거 있는 의사결정을 하고 싶다
- 임원 보고용 시각화 자료를 빠르게 만들고 싶다

**Pain Points:**
- 분석 요청마다 개발팀·데이터팀에 의존
- 엑셀 피벗이 한계에 달했을 때 대안 없음
- BI 툴(Tableau, Power BI)은 너무 복잡하고 비쌈

---

#### 세그먼트 B: 컨설팅·에이전시 팀 (성장)

| 항목 | 내용 |
|------|------|
| 기업 유형 | 경영 컨설팅, 마케팅 에이전시, 리서치 기관 |
| 기업 규모 | 5~100명 |
| 역할 | 컨설턴트, 애널리스트, 어카운트 매니저 |
| 기술 수준 | 중간 (엑셀 고급, 일부 SQL) |
| 예산 | 월 20~100만 원 |
| 주요 니즈 | 클라이언트별 분석 속도, 발표 자료 품질 |

**Jobs to Be Done:**
- 클라이언트 데이터를 빠르게 분석해 인사이트를 제공하고 싶다
- PPT 발표용 차트를 자동 생성하고 싶다
- 여러 클라이언트를 동시에 효율적으로 관리하고 싶다

**Pain Points:**
- 클라이언트마다 데이터 포맷이 달라 정리에 시간 소요
- 마감 압박 속에서 분석 품질 유지 어려움
- 인력 대비 프로젝트 수 증가로 생산성 한계

---

#### 세그먼트 C: 1인 사업자·프리랜서 (볼륨)

| 항목 | 내용 |
|------|------|
| 유형 | 온라인 셀러, 유튜버, 마케터, 소규모 사업주 |
| 규모 | 1~5인 |
| 기술 수준 | 낮음 (엑셀 기초 수준) |
| 예산 | 월 1~10만 원 |
| 주요 니즈 | 셀프서비스, 즉각적 결과 |

**Jobs to Be Done:**
- 내 사업 데이터가 어떤 의미인지 알고 싶다
- 복잡한 툴 없이 바로 쓸 수 있는 분석 도구가 필요하다

**Pain Points:**
- 데이터 분석 전문 지식 부족
- 고가 툴 구독 부담
- 분석 결과를 어떻게 해석해야 할지 모름

---

#### 세그먼트 D: 스타트업 내부 데이터팀 (미래)

| 항목 | 내용 |
|------|------|
| 기업 규모 | 시리즈 A~B, 50~300명 |
| 역할 | 데이터 분석가, PM, 그로스 팀 |
| 기술 수준 | 높음 (SQL, Python 가능) |
| 예산 | 월 50~200만 원 |
| 주요 니즈 | 자동화, 협업, 재현 가능한 분석 |

**Pain Points:**
- 반복 분석 자동화 필요
- 비기술 팀원들과의 데이터 공유 어려움
- 내부 대시보드 구축 리소스 부족

## 2. 타겟 선정 (Targeting)

### 세그먼트 평가 매트릭스

| 평가 기준 | 가중치 | A: 중소기업팀 | B: 에이전시 | C: 1인 사업자 | D: 스타트업팀 |
|-----------|--------|--------------|------------|--------------|--------------|
| 시장 규모 | 20% | 9 | 7 | 10 | 6 |
| 성장률 | 20% | 8 | 8 | 7 | 9 |
| 수익성 | 20% | 7 | 8 | 4 | 9 |
| 경쟁 강도 | 15% | 6 | 7 | 5 | 4 |
| 제품 적합성 | 15% | 9 | 8 | 8 | 6 |
| 접근 용이성 | 10% | 8 | 7 | 9 | 5 |
| **가중 점수** | | **7.65** | **7.55** | **6.65** | **6.50** |

### 세그먼트별 강약점

| 세그먼트 | 강점 | 약점 |
|----------|------|------|
| A 중소기업팀 | 명확한 Pain, 높은 WTP, 팀 단위 구독 | B2B 영업 사이클 존재 |
| B 에이전시 | 멀티시트 확장, 레퍼런스 효과 | 클라이언트 이탈 시 구독 해지 위험 |
| C 1인 사업자 | 자기서비스, 대규모 트래픽 | ARPU 낮음, 이탈률 높음 |
| D 스타트업팀 | 높은 ARPU, 장기 계약 | 긴 평가 기간, 경쟁사 다수 |

### 타겟 전략: 집중화 → 차별화 순차 확장

**MVP 1차 타겟 (집중화): 세그먼트 A — 중소기업 마케팅·운영팀**
- DeepMetria 핵심 가치(비개발자 대상 AI 분석)와 가장 높은 적합성
- 자기서비스 가능, 빠른 가입 결정
- Pro/Max 업그레이드 경로가 명확

**2차 타겟 (확장): 세그먼트 B — 컨설팅·에이전시 팀**
- 팀 단위 구독으로 ARPU 극대화
- 고객사 소개를 통한 바이럴 효과 기대

**서비스하되 집중하지 않음: 세그먼트 C**
- Free 티어로 유입, 자연 전환 유도
- 리소스 집중 대상 아님

**보류: 세그먼트 D**
- MVP 이후 Pro/Max 안정화 후 진입

## 3. 포지셔닝 (Positioning)

### 경쟁 포지셔닝 맵

```
                  사용 편의성 (높음)
                        │
           DeepMetria   │
           (목표)       │   ChatGPT
                        │   (일반형)
                        │
비전문가 ───────────────┼─────────────── 전문가
친화적                  │               친화적
                        │
           Google       │   Tableau/
           Looker Studio│   Power BI
                        │
                  사용 편의성 (낮음)
              (분석 전문성 요구)
```

### 포지셔닝 옵션

**옵션 1: 비개발자의 데이터 분석가**
```
데이터를 분석하고 싶지만 코딩을 모르는 비즈니스 팀을 위해,
DeepMetria는 LLM 기반 데이터 분석 플랫폼으로,
자연어 질문만으로 전문가 수준의 분석과 시각화를 제공합니다.
기존 BI 툴(Tableau, Power BI)과 달리,
저희는 코드나 SQL 없이 대화하듯 데이터와 대화하게 합니다.
```

**옵션 2: AI 데이터 팀장**
```
데이터 분석가를 고용할 여력이 없는 중소기업 팀을 위해,
DeepMetria는 AI 오케스트레이터 기반 분석 SaaS로,
파일 업로드부터 대시보드, 발표 자료까지 원스톱으로 처리합니다.
ChatGPT나 엑셀과 달리,
저희는 분석 도구를 자동 선택하고 맥락 있는 인사이트를 제공합니다.
```

**옵션 3: 분석의 민주화**
```
데이터 분석이 일부 전문가만의 영역이라고 생각하는 팀을 위해,
DeepMetria는 누구나 쓸 수 있는 AI 분석 플랫폼으로,
"왜 매출이 떨어졌나요?"처럼 자연어로 물으면 답을 드립니다.
복잡한 BI 툴과 달리,
저희는 5분 안에 첫 인사이트를 경험할 수 있습니다.
```

### 최종 포지셔닝 권고

**1차 타겟(중소기업팀) 기준 권고 포지셔닝:**

> **DeepMetria는 비개발자를 위한 AI 데이터 분석 플랫폼입니다.**
>
> 데이터를 올리고 궁금한 것을 물어보면, AI가 알아서 분석하고 시각화합니다.
> SQL도, 코딩도, 복잡한 설정도 필요 없습니다.
>
> - 파일 업로드 → 자동 데이터 요약
> - 자연어 질문 → 적합한 분석 + 차트 자동 선택
> - PPT처럼 편집 가능한 대시보드 → 바로 다운로드
>
> Tableau는 너무 어렵고, ChatGPT는 시각화가 안 됩니다.
> DeepMetria는 그 사이를 채웁니다.

### 포지셔닝 증거 포인트

| 주장 | 증거/지표 |
|------|----------|
| 비개발자도 즉시 사용 가능 | 가입 후 5분 내 첫 인사이트 달성 |
| 자동 분석 도구 선택 | 사용자가 차트 유형을 지정할 필요 없음 |
| 발표 자료 즉시 활용 | PPT/PDF 다운로드, 서식 편집 지원 |
| 깊이 있는 분석 | Chain of Thinking 기반 다단계 추론 |
| 멀티 LLM 지원 | Claude, GPT, Gemini 선택 가능 |

### 과금 티어별 포지셔닝

| 티어 | 타겟 | 핵심 메시지 |
|------|------|------------|
| Free | 1인 사업자, 개인 | "데이터 분석, 일단 무료로 시작하세요" |
| Pro | 중소기업 담당자 | "팀의 데이터 분석을 자동화하세요" |
| Max | 에이전시, 다수 팀 | "클라이언트마다 전문 분석 보고서를" |
